Add custom post type Salon Report, taxonomy, and field.

== Description ==

Salon Report plugin is add custom post type.

= Features =

* Add custom post type.
* Add custom taxonomy.
* Add custom field.
* Add hand drawing field.

== Installation ==

1. Upload `Salon Report` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. You can setting post type Salon Report.
